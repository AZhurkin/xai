# Задача:
На основе входной записи сформировать **парафразный (переформулированный) вопрос на русском языке**,
полностью эквивалентный исходному вопросу по смыслу и проверяемый тем же самым ответом и evidence span.

# Входные данные (одна строка `jsonl`):
```json
{"question": "...", "answer": "...", "evidence_span": "...", "context": "..."}
```

# Инструкции:
1. Внимательно прочитай и проанализируй:
   * исходный вопрос (`question`);
   * соответствующий ему ответ (`answer`);
   * `evidence_span` — дословную цитату из текста, доказывающую корректность ответа;
   * `context` — исходный текст, из которого были получены вопрос, ответ и evidence span.
2. Сформулируй **новый вопрос**, который:
   * является **парафразом исходного вопроса**, то есть полностью эквивалентен ему по смыслу;
   * предполагает **тот же самый ответ**, подтверждаемый **тем же самым evidence span**;
   * не изменяет информационный фокус и не смещает условия вопроса.
3. При переформулировании вопроса соблюдай следующие требования:
   * измени лексику (используй синонимы), порядок слов и/или грамматическую конструкцию;
   * не добавляй новых условий, ограничений или уточнений;
   * не убирай существенные условия, присутствующие в исходном вопросе;
   * не делай вопрос более общим или более узким по смыслу;
   * избегай дословного копирования формулировок исходного вопроса;
   * допускается использование ключевых терминов из текста, если без них теряется точность;
   * вопрос должен быть формальным и нейтральным по стилю;
   * вопрос полностью на русском языке.
4. **Запрещено:**
   * использовать информацию, отсутствующую в `context`;
   * формулировать вопрос, требующий иного ответа или иного evidence span;
   * превращать вопрос в многочастный или неоднозначный;
   * добавлять пояснения, комментарии или мета-текст.
5. Тип вопроса всегда указывай как `"paraphrasing"`.
6. **Формат вывода строго `jsonl`**, без дополнительного текста.
**Формат вывода:**
```json
{"question": "...", "type": "paraphrasing"}
```
