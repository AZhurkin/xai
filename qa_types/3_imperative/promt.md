# Задача:
На основе предоставленного фрагмента текста на русском языке сформировать пары *вопрос–ответ*
**в императивном стиле (команда/просьба)** с обязательным указанием *evidence_span* для доказательства корректности ответа,
**не менее 3 пар *вопрос–ответ***, каждая из которых основана на **разных фактах**!

# Инструкции:
1. Внимательно прочитай исходный текст на русском языке и используй **исключительно**
   информацию, содержащуюся в нём.
2. Выбери **как минимум три различных явно сформулированных факта, правила или утверждения**,
   присутствующих в тексте и подходящих для проверки корректности извлечения информации.
   * Каждый факт должен быть независимым от остальных.
3. Для **каждого факта отдельно** сформулируй вопрос и ответ.
4. Сформулируй **вопрос в императивном наклонении** (команда/просьба), соблюдая следующие требования:
   * вопрос должен быть формальным и нейтральным, без разговорных частиц и оценочных формулировок;
   * используй глаголы (или эквиваленты): **«перечислите», «назовите», «укажите», «определите», «объясните», «сформулируйте», «опишите», «выделите», «найдите»**;
   * сформулируй запрос так, чтобы ожидаемый ответ был **кратким и структурированным** (например: «Перечислите пункты…», «Укажите требования…»);
   * вопрос должен быть однозначным и проверяемым по `evidence_span`;
   * вопрос полностью на русском языке.
5. Сформулируй **краткий и точный ответ**, строго на основе текста:
   * Ответ отражает только информацию, явно присутствующую в документе.
   * Запрещено добавлять домыслы, интерпретации или внешние знания.
   * Допускается краткое переформулирование без искажения смысла.
   * Ответ не должен быть избыточным или содержать длинные цитаты.
   * Ответ полностью на русском языке.
   * Ответ однозначно подтверждается evidence span.
6. **Evidence span**:
   * `evidence_span` — это **дословная цитата из исходного текста**, которая **непосредственно доказывает корректность ответа**.
   * Evidence span должен быть **непрерывным фрагментом текста**.
   * Цитата должна быть **минимально достаточной** для подтверждения ответа.
   * Запрещено:
     * изменять текст цитаты;
     * сокращать её многоточиями;
     * объединять несколько несмежных фрагментов;
     * добавлять пояснения вне цитаты.
7. Поле `"type"` **всегда** заполняй значением `"imperative"`.
8. **Формат вывода строго `jsonl`**, без комментариев и дополнительного текста.
   Каждая строка — один объект.

# Формат:
```json
{"question": "...", "answer": "...", "evidence_span": "...", "type": "imperative"}
```

# Ограничения:
    * Один вопрос — один факт.
    * Никакой информации вне текста.
    * Строгое соблюдение формата.
